# It is an AI - Powered Adaptive Learning Platform for Physic Enthusiasts.

PHO-Guide is an innovative web application dsigned to help highschool and college students prepare for phyics competitions like IPhO, national olympics, and advanced exams. Built by passionate high school physics enthusiasts, it combines cutting-edge AI with proven educational techniques to deliver personalized, adaptive learning experiences.

The program is open source under MIT license.
